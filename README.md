# AI-Powered Fitness Buddy

## Overview
AI-Powered Fitness Buddy is an Agentic AI project developed using IBM watsonx Orchestrate.

The assistant provides beginner-friendly workout plans, healthy eating guidance, hydration tips, muscle gain suggestions, weight loss recommendations, and recovery advice using a Retrieval-Augmented Generation (RAG) approach with an uploaded Fitness Knowledge Base.

## Technologies Used
- IBM watsonx Orchestrate
- IBM Cloud
- Retrieval-Augmented Generation (RAG)
- IBM Foundation Model

## Features
- Beginner workout plans
- Healthy eating guidance
- Hydration advice
- Muscle gain tips
- Weight loss recommendations
- Recovery guidance
- Knowledge-base-driven responses
